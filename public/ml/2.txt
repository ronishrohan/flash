ML Program 2: Correlation Matrix, Heatmap, and Pair Plot

Aim:
Compute correlation between features and visualize relationships using heatmap and pair plot.

Install:
pip install pandas matplotlib seaborn scikit-learn

Steps:
1. Load California Housing dataset.
2. Compute correlation matrix using df.corr().
3. Plot heatmap.
4. Plot pair plot for feature relationships.

Code:

import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.datasets import fetch_california_housing

data = fetch_california_housing(as_frame=True)
df = data.frame

print(df.head())

corr = df.corr()
print("\nCorrelation matrix:")
print(corr)

plt.figure(figsize=(10, 8))
sns.heatmap(corr, annot=True, cmap="coolwarm", fmt=".2f")
plt.title("Correlation Heatmap")
plt.show()

# Pair plot can be slow, so use a small sample.
sample = df.sample(500, random_state=42)
sns.pairplot(sample, corner=True)
plt.show()

Expected output:
- Correlation matrix values.
- Heatmap showing strong positive/negative relationships.
- Pair plot for sampled data.

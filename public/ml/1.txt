ML Program 1: Histograms and Box Plots - California Housing

Aim:
Create histograms and box plots for all numerical features and identify outliers.

Install:
pip install pandas numpy matplotlib seaborn scikit-learn

Steps:
1. Load California Housing dataset.
2. Convert it to a DataFrame.
3. Plot histograms for numerical feature distribution.
4. Plot box plots for outlier visibility.
5. Use IQR rule to count outliers.

Code:

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.datasets import fetch_california_housing

data = fetch_california_housing(as_frame=True)
df = data.frame

print(df.head())

df.hist(figsize=(12, 8), bins=30, edgecolor="black")
plt.suptitle("Histograms of California Housing Features")
plt.tight_layout()
plt.show()

plt.figure(figsize=(12, 8))
for i, col in enumerate(df.columns, 1):
    plt.subplot(3, 3, i)
    sns.boxplot(y=df[col])
    plt.title(col)
plt.tight_layout()
plt.show()

print("\nOutlier count using IQR:")
for col in df.columns:
    q1 = df[col].quantile(0.25)
    q3 = df[col].quantile(0.75)
    iqr = q3 - q1
    lower = q1 - 1.5 * iqr
    upper = q3 + 1.5 * iqr
    count = ((df[col] < lower) | (df[col] > upper)).sum()
    print(col, ":", count)

Expected output:
- Dataset first rows.
- Histograms.
- Box plots.
- Outlier count for every feature.

ML Program 10: K-Means Clustering on Breast Cancer Dataset

Aim:
Apply K-Means clustering on Wisconsin Breast Cancer dataset and visualize clusters.

Install:
pip install matplotlib scikit-learn

Steps:
1. Load Breast Cancer dataset.
2. Standardize features.
3. Apply K-Means with 2 clusters.
4. Reduce to 2D using PCA for visualization.
5. Plot clusters.

Code:

import matplotlib.pyplot as plt
from sklearn.datasets import load_breast_cancer
from sklearn.preprocessing import StandardScaler
from sklearn.cluster import KMeans
from sklearn.decomposition import PCA

data = load_breast_cancer()
X = data.data

X_scaled = StandardScaler().fit_transform(X)

kmeans = KMeans(n_clusters=2, random_state=42, n_init=10)
clusters = kmeans.fit_predict(X_scaled)

pca = PCA(n_components=2)
X_2d = pca.fit_transform(X_scaled)

plt.scatter(X_2d[:, 0], X_2d[:, 1], c=clusters, cmap="viridis")
plt.xlabel("PC1")
plt.ylabel("PC2")
plt.title("K-Means Clustering on Breast Cancer Dataset")
plt.show()

Expected output:
- 2D cluster plot showing two clusters.
